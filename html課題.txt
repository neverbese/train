○言語
・HTML
・CSS
・JavaScript

※CSSやJavaScriptはフレームワークOKです。
※但し、それぞれ1種類以内でお願いします。

○フォルダ構成

workspace ┬ css
├ data
├ img
├ js
└ 各種HTML

・作業フォルダ（workspace）の配下に4つのフォルダを置きます。
（css / data / img / js）

○ファイル構成

・HTML：
作業フォルダの直下に置いてください。
ページの個数分のファイルができると思います。

・CSS：
cssフォルダ配下に置いてください。
1個にまとめても、複数に分かれても構いません。

・JavaScript
jsフォルダ配下に置いてください。
1個にまとめても、複数に分かれても構いません。

・画像データ
imgフォルダ配下に置いてください。
使わなくても構いません。

・一覧のデータ   
dataフォルダ配下に置いてください。
JSON形式で作成し、それをJavaScriptで読み込んで
一覧のページを作成できるといいなと思っています。
難しそうなら相談してください。

・IDとパスワードの照合データ
dataフォルダ配下に置いてください。
JSON形式で作成し、それをJavaScriptで読み込んで
入力されたIDとパスワードと照合するイメージです。

○ファイル構成（ツリー）

workspace

○ページ構成

・ログインページ
参考サイトにはありません。
いわゆるIDとパスワードを入力して
ログインさせるシンプルな作りを想定しています。

IDとパスワードの照合も、
dataフォルダ配下にマッチングデータを作り
それを見に行くようにしましょうか！
（DBも使いません）

・トップページ
参考サイトと同じです。
経費登録ページに飛べるボタンと
経費照会ページに飛べるボタンがあればいいです。
サンプルのダウンロードは要りません。

・経費登録ページ
参考サイトと同じです。
経費を登録します。
一か所、プルダウンからの選択があるといいですね。
経費登録すると一覧に溜まっていきますが、
セッションに保持させるような形で実現できるでしょうか。
（DBは使いません、ブラウザを閉じたら消えてOKです）

・経費照会ページ
参考サイトと同じです。
10件ずつ、ページングさせてください。
実は、取引処理番号に詳細ページのリンクがあります。
一旦、ここはなしで構いません。
（余力があれば、チャレンジしてください）
